1. How to create new notification:

	1.1. Open incident page
	1.2. Press ALT+5
	1.3. Run "Template Wizard.exe"
	1.4. Select "Create New"
	1.5. Notification is ready in "files" folder in "output.docx"
	
	Remarks: pressing ALT+5 copies all necessary data to clipboard, don't populate your clipboard with different data before selecting "Create New"
		 or the programm won't work.



2. How to change notification color:
	
	2.1. Go to "files" folder
	2.2. Open "color_change.docx"
	2.3. On clear 1st page paste the notification that requires color change and save the document
	2.4. Run "Template Wizard.exe"
	2.5. Select "Change Color of existing one"
	2.6. Select color
	2.7. Notification is ready in "files" folder in "output.docx"
